Flutter Widgets Complete Learning Guide

Flutter UI is built using widgets. Everything visible on the screen is a widget. This guide explains the most important widgets every beginner should learn.

1. Scaffold

Scaffold provides the basic layout of a screen.

Scaffold(
 appBar: AppBar(title: Text("Home")),
 body: Center(
  child: Text("Hello Flutter")
 )
)
Property Description
appBar Top navigation bar
body Main content area
drawer Side navigation menu
floatingActionButton Floating button

2. AppBar

AppBar is the top bar of an application.

AppBar(
 title: Text("Dashboard"),
 centerTitle: true,
 actions: [
  Icon(Icons.search)
 ]
)

3. Container

Container is used for layout, styling and positioning.

Container(
 width:200,
 height:100,
 color:Colors.blue,
 child: Text("Hello")
)
Property Purpose
width Set container width
height Set container height
padding Space inside container
margin Space outside container

4. Text Widget

Used to display text.

Text(
 "Hello Flutter",
 style: TextStyle(
  fontSize:20,
  fontWeight:FontWeight.bold
 )
)

5. Row

Row arranges widgets horizontally.

Row(
 children:[
  Icon(Icons.home),
  Text("Home")
 ]
)

6. Column

Column arranges widgets vertically.

Column(
 children:[
  Text("Title"),
  Text("Subtitle")
 ]
)

7. Padding

Adds space around a widget.

Padding(
 padding: EdgeInsets.all(20),
 child: Text("Hello")
)

8. SizedBox

Creates fixed spacing.

SizedBox(height:20)

9. Image

Displays images.

Image.asset("assets/image.png")

Image.network("https://example.com/image.png")

10. ListView

Scrollable list.

ListView(
 children:[
  Text("Item 1"),
  Text("Item 2")
 ]
)

ListView.builder(
 itemCount:10,
 itemBuilder:(context,index){
  return Text("Item $index");
 }
)

11. Stack

Stack allows widgets to overlap.

Stack(
 children:[
  Image.asset("bg.png"),
  Text("Overlay Text")
 ]
)

12. Buttons

ElevatedButton

ElevatedButton(
 onPressed:(){},
 child:Text("Submit")
)

OutlinedButton

OutlinedButton(
 onPressed:(){},
 child:Text("Cancel")
)

TextButton

TextButton(
 onPressed:(){},
 child:Text("Skip")
)

13. TextField

User text input field.

TextField(
 decoration: InputDecoration(
  hintText:"Enter Name"
 )
)

14. GestureDetector

Detects user gestures.

GestureDetector(
 onTap:(){
  print("Tapped");
 },
 child: Container(
  color: Colors.blue,
  height:50
 )
)

15. SafeArea

Prevents UI from overlapping system areas.

SafeArea(
 child: Text("Safe Content")
)

16. MediaQuery

Gets screen size.

double width = MediaQuery.of(context).size.width;
double height = MediaQuery.of(context).size.height;

17. Navigation

Move between screens.

Navigator.push(
 context,
 MaterialPageRoute(
  builder:(context)=>NextScreen()
 )
)
