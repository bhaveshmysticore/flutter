Flutter Complete Learning Guide

This documentation explains Flutter widgets from beginner to advanced level. Flutter builds UI using widgets. Everything you see in a Flutter application is a widget.

1. What is Flutter?

Flutter is an open-source UI framework developed by Google used to build mobile, web, and desktop applications from a single codebase.

Flutter uses the Dart programming language and follows a widget-based architecture.

2. Core Widgets

Text Widget

Displays text on screen.

Text(
 "Hello Flutter",
 style: TextStyle(
  fontSize:20,
  fontWeight:FontWeight.bold
 )
)

Container Widget

Container is used to control layout, styling, and spacing. 

Container(
 width:200,
 height:100,
 color:Colors.blue,
 child:Text("Hello")
)

3. Layout Widgets

Row

Row arranges widgets horizontally.

Row(
 children:[
  Icon(Icons.star),
  Text("Rating")
 ]
)

Column

Column arranges widgets vertically.

Column(
 children:[
  Text("Title"),
  Text("Description")
 ]
)

Stack

Stack places widgets on top of each other.

Stack(
 children:[
  Image.asset("bg.png"),
  Text("Overlay Text")
 ]
)

6. List Widgets

ListView

ListView(
 children:[
  Text("Item 1"),
  Text("Item 2")
 ]
)

GridView

GridView.count(
 crossAxisCount:2,

  Container(color:Colors.red),

7. Navigation

Navigation is used to move between screens. 

Navigator.push(
 context,
 MaterialPageRoute(
  builder:(context)=>NextScreen()
 )
)

8. Responsive Design

MediaQuery helps create responsive UI. 

double width = MediaQuery.of(context).size.width;
double height = MediaQuery.of(context).size.height;
