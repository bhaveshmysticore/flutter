Flutter Widgets Complete Learning Guide (Beginner → Professional)

Flutter applications are built completely using widgets. A widget represents a part of the user interface such as text, image, button, layout, or screen structure.

This guide explains the most important Flutter widgets with examples so beginners can understand how UI is built.

1. Scaffold

What is Scaffold?

Scaffold is the main layout structure of a Flutter screen. Almost every page in a Flutter app uses Scaffold because it provides common UI elements like:

  • AppBar (top navigation bar)
  • Body (main screen content)
  • Drawer (side menu)
  • Floating Action Button
  • Bottom Navigation Bar

Example

Scaffold(
 appBar: AppBar(
  title: Text("Home Screen")
 ),
 body: Center(
  child: Text("Welcome to Flutter")
 )
)

Important Properties

Property Purpose
appBar Top navigation bar
body Main screen UI
drawer Side navigation menu
floatingActionButton Floating button for quick actions
bottomNavigationBar Bottom navigation tabs

2. Container

Purpose

Container is one of the most commonly used widgets. It works like a box that can contain other widgets and allows you to control layout and styling.

Common Uses

  • Adding background color
  • Setting width and height
  • Adding padding and margin
  • Applying borders and radius

Example

Container(
 width:200,
 height:100,
 color:Colors.blue,
 child:Center(
  child:Text("Hello")
 )
)

Important Properties

Property Description
width Defines container width
height Defines container height
padding Space inside container
margin Space outside container
decoration Styling such as border and radius

3. Text Widget

Purpose

Text widget displays text on the screen. It is used for titles, descriptions, labels, buttons, and messages.

Example

Text(
 "Hello Flutter",
 style: TextStyle(
  fontSize:20,
  fontWeight:FontWeight.bold,
  color:Colors.black
 )
)

Important Properties

Property Purpose
style Controls text appearance
textAlign Text alignment
maxLines Limit text lines
overflow Handle long text overflow

4. Row

Purpose

Row widget arranges its children horizontally.

Example

Row(
 children:[
  Icon(Icons.star),
  Text("Rating")
 ]
)

Important Properties

Property Description
mainAxisAlignment Controls horizontal spacing
crossAxisAlignment Controls vertical alignment

5. Column

Purpose

Column arranges widgets vertically.

Example

Column(
 children:[
  Text("Title"),
  Text("Subtitle")
 ]
)

6. Padding

Padding adds space around a widget inside its boundary. 

Padding(
 padding:EdgeInsets.all(20),
 child:Text("Hello")
)

7. SizedBox

SizedBox is used to create empty space between widgets or set fixed width/height. 

SizedBox(height:20)

8. Image Widget

Image widget is used to display images in Flutter applications.

Asset Image

Image.asset("assets/images/logo.png")

Network Image

Image.network("https://example.com/image.png")

9. ListView

ListView displays scrollable lists of items.

Static List

ListView(
 children:[
  Text("Item 1"),
  Text("Item 2"),
  Text("Item 3")
 ]
)

Dynamic List

ListView.builder(
 itemCount:10,
 itemBuilder:(context,index){
  return Text("Item $index");
 }
)

10. SafeArea

SafeArea ensures UI elements do not overlap with system areas like:

  • Status bar
  • Notch
  • Gesture navigation area
SafeArea(
 child:Text("Safe Content")
)

11. GestureDetector

GestureDetector detects user interactions such as taps or long presses. 

GestureDetector(
 onTap:(){
  print("Tapped");
 },
 child:Container(
  color:Colors.blue,
  height:50,
  width:100
 )
)

12. Navigation

Navigation allows moving between different screens. 

Navigator.push(
 context,
 MaterialPageRoute(
  builder:(context)=>NextScreen()
 )
)

Basic Flutter Layout Structure

MaterialApp
   |
Scaffold
   |
SafeArea
   |
Column / Row
   |
Container
   |
Text / Image / Button

Most Flutter applications follow this structure when building screens.
