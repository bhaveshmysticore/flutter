Flutter Complete Learning Guide

This guide explains Flutter from beginner to advanced level. Flutter builds UI using widgets. Every UI element in Flutter is a widget.

Goal of this guide:

  • Understand Flutter architecture
  • Learn important widgets
  • Build responsive layouts
  • Handle forms and user input
  • Implement navigation
  • Create animations

1. Flutter Application Structure

void main(){
 runApp(MyApp());
}

class MyApp extends StatelessWidget{

 @override
 Widget build(BuildContext context){

  return MaterialApp(
   title:"Flutter App",
   home:HomeScreen()
  );

 }

}

Explanation:

  • main() → Starting point of Flutter application
  • runApp() → Loads root widget
  • MaterialApp → Main app configuration

2. Scaffold Widget

Scaffold provides the basic layout structure of a screen. 

Scaffold(
 appBar:AppBar(
  title:Text("Home")
 ),
 body:Center(
  child:Text("Welcome")
 ),
 floatingActionButton:FloatingActionButton(
  onPressed:(){},
  child:Icon(Icons.add)
 )
)

Main Properties

Property Purpose
appBar Top navigation bar
body Main screen content
drawer Side navigation menu
floatingActionButton Main action button
bottomNavigationBar Bottom navigation tabs

3. Container Widget

Container is used to control layout and styling. 

Container(
 width:200,
 height:120,
 padding:EdgeInsets.all(10),
 decoration:BoxDecoration(
  color:Colors.blue,
  borderRadius:BorderRadius.circular(10)
 ),
 child:Text("Hello Flutter")
)

Important Properties

PropertyDescription
widthContainer width
heightContainer height
paddingInner spacing
marginOuter spacing
decorationStyling

6. Column Widget

Column arranges widgets vertically. 

Column(
 children:[
  Text("Title"),

 ]
)

7. Stack Widget

Stack places widgets on top of each other. 

Stack(
 children:[
  Image.asset("bg.png"),
  Positioned(
   bottom:10,
   child:Text("Overlay Text")
  )
 ]
)

14. MediaQuery (Responsive Design)

MediaQuery helps build responsive layouts.

15. Form and Validation

Form(
 key:_formKey,
 child:Column(
  children:[

   TextFormField(
    validator:(value){
     if(value!.isEmpty){
      return "Enter name";
     }
     return null;
    }
   ),

   ElevatedButton(
    onPressed:(){
     if(_formKey.currentState!.validate()){
      print("Valid");
     }
    },
    child:Text("Submit")
   )

  ]
 )
)
